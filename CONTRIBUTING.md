Teşekkürler! Katkıda bulunmak isterseniz lütfen aşağıdaki adımları izleyin:

- Fork yapın ve feature branch oluşturun: `git checkout -b feature/isim`
- Değişiklikleri küçük, mantıklı commit'lerle gönderin.
- Kod yazmadan önce issue açın veya mevcut bir issue'ya atayın.
- Pull request oluştururken değişikliklerin kısa özetini ve nasıl test edildiğini yazın.

Local test ve lint adımları:

```bash
python -m venv .venv
pip install -r requirements.txt
# Daha sonra örnek olarak notebook'u açıp çalıştırabilirsiniz
```
